# Frontend del Portafolio

Este directorio contiene el código fuente de la interfaz web del portafolio de Ingeniería de Datos.

Para obtener información detallada sobre el proyecto completo, la arquitectura, el stack tecnológico y las instrucciones de instalación, consulte el **[README principal](file:///c:/Users/Sr.%20Fall/Desktop/Data%20Git/data_porfolio/README.md)** en la raíz del repositorio.

---

## 🚀 Inicio Rápido (Frontend)


1. Instalar dependencias: `npm install`
2. Iniciar servidor de desarrollo: `npm run dev`
3. Construir para producción: `npm run build`
